ЦЕЛЬ:

Понять, что такое функции, зачем они нужны и как их использовать.

ЧТО ТАКОЕ ФУНКЦИЯ?

Функция — это блок кода, который можно вызвать в любой момент, передав в него аргументы. Она помогает избегать дублирования кода.

Определение функции

Функция определяется с помощью ключевого слова def, за которым идет имя функции и список параметров. После этого идет тело функции, которое выполняется при ее вызове.

Шаги для создания функции:

  • Шаг 1: Напишите ключевое слово def, за которым идет имя функции.
  • Шаг 2: Введите параметры функции в круглых скобках.
  • Шаг 3: Напишите двоеточие после списка параметров.
  • Шаг 4: Определите тело функции, отступив на 4 пробела.
  • Шаг 5: Вызовите функцию, передав ей необходимые аргументы.
# Шаг 1: Напишите ключевое слово 'def'
def имя_функции(параметры):
    # Шаг 4: Определите тело функции
    # Тело функции с нужной логикой
    print("Пример функции")

# Шаг 5: Вызовите функцию с аргументами
имя_функции(аргумент)

# Пример: создание и вызов функции

# Шаг 1: Определение функции с параметром
def приветствие(имя):
    # Шаг 4: Тело функции
    print("Привет,", имя, "!")

# Шаг 5: Вызов функции с аргументом
приветствие("Анна")
приветствие("Иван")

# Функция с возвратом значения
def квадрат(число):
    # Шаг 4: Тело функции с возвратом значения
    return число * число

результат = квадрат(5)
print("Квадрат числа 5:", результат)

# Функция с несколькими параметрами
def сумма(а, б):
    # Шаг 4: Тело функции с несколькими параметрами
    return а + б

print("Сумма 3 и 7:", сумма(3, 7))

Что нужно запомнить:

  • Функции упрощают код и делают его читаемым.
  • return используется для возврата результата.
  • Функция может принимать аргументы, которые передаются при вызове.
